# Proposito del proyecto
Este proyecto busca clasificar tumores cerebrales (meningionmas, gliomas, tumores pituitarios y cerebros sanos) en imagenes de resonancias mageneticas. Para cumplir con este objetivo se hara uso tanto de redes neuronales superficiales, asi como de redes neuronales profundas con el fin de evaluar sus desempeños en este problema en particular.
