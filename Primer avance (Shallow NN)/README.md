# Primer avance (Shallow NN)
En este primer avance se usa una arquitectura de 3 capas definidas de la siguiente manera:

## Capa de entrada
* Tamaño: 24 valores

## Capa oculta
* Tamaño: 100 neuronas
* Funcion de activacion: ReLU

## Capa de salida
* Tamaño: 4 valores
* Funcion de activacion: Softmax

# Ejecucion del notebook
Se recomienda ejecutar este notebook usando jupyter notebook de forma local. Antes de ejecutar el notebook se debe usar el siguiente comando para instalar las librerias necesarias:

```code
pip install -r requirements.txt
```
